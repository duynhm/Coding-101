## Program Description
- **Purpose**: Calculate the sum of integers from 1 to n (S = 1 + 2 + 3 + ... + n)
- **Input**: n = 10
- **Process**: Uses a while loop to iterate from 0 to n, accumulating the sum
- **Output**: Prints the sum (55 for n=10)
